Инструкция:
- Установить postgresql версии 16+
- Скачать из репозитория архив с исходным кодом / клонировать репозиторий с помощью git clone
- Установить зависимости requirements.txt с помощью pip
- Распаковать, указать в .env корректные значения для подключения к бд и нужный порт для фронтенда, и запустить
